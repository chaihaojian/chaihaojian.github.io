关键字

关键字是C++中预先保留的单词（标识符），在定义变量或常量时，不要用关键字，否则会产生歧义。

标识符命名规则

  1. 标识符不能是关键字；

  2. 标识符只能由字母、数字、下划线组成；

  3. 第一个字符必须为字母或下划线；

  4. 标识符中的字母区分大小写；

数据类型（给变量或常量分配合适的内存空间）

C++规定在创建一个变量或者常量时，必须要指定相应的数据类型，否则无法给变量分配内存。

实型（浮点型）

浮点型变量分为两种：

  1. 单精度 float
  2. 双精度 double

两者的区别在于表示的有效数字范围不同。

数据类型 占用空间 有效数字范围
float 4字节 7位有效数字
double 8字节 15-16位有效数字 
1float a = 3.14f;//(float)(3.14F) 建议使用该方法声明一个float变量
2float b = 3.14;//(double)(3.14)

上述两种方法都声明了一个float变量，但第二种方法c++默认值小数3.14是double类型，在声明后需要将double类型的3.14转换为float类型，在变量值后加f则表明3.14是float类型；

1float a = 3.1415926f;
2float b = 3.14;
3

5
6double m = 3.1415926;
7cout << "m = " << m << endl;

注意：默认情况下，float和double类型的变量在输出时，都只会显示出6位有效数字。

数组

数组的特点

  1. 数组中的每一个数据元素都是相同的数据类型；
  2. 数组是由连续的内存位置组成的；

数组的定义

  1. 数据类型 数组名[数组长度]；
  2. 数据类型 数组名[数组长度] = {值1，值2 …};
  3. 数据类型 数组名[] = {值1，值2 …};

一维数组

一维数组在内存中的长度

1int arr[5] = {1,2,3,4,5};
2cout << "arrdress of arr: " << arr << endl;
3cout << "size of arr: " << sizeof(arr) << endl;
4cout << "size of arr[0]: " << sizeof(arr[0]) << endl;
5cout << "length of arr: " << sizeof(arr) / sizeof(arr[0]) << endl;

一维数组的冒泡排序

思路（升序）：

  1. 比较相邻的两个元素，如果第一个比第二个大，就交换它们的位置；
  2. 对每一个元素做同样的工作，执行完第一次后，找到数组中的第一个最大值；
  3. 重复上述步骤，每次比较次数-1，直到不需要比较；

指针

作用：可以通过指针直接访问内存

  • 内存编号是从0开始记录的，一般用16进制数字表示；

  • 可以利用指针变量保存地址；

指针变量的定义和使用

  • 定义语法：数据类型 * 变量名；

  • 使用方法：可以通过解引用的方式来找到指针指向的内存中保存的值；指针变量前加*代表解引用，找到指针指向的内存中的数据；

 1int main()
 2{
 3  int a = 10;
 4  int * p;
 5  p = &a;
 6  cout << "the value of a: " << p << endl;
 7  cout << "address of a: " << &a << endl;
 8  cout << "the value of p: " << p << endl;
 9  cout << "the value pointed to by pointer p: " << *p << endl;
10
11  *p = 100;
12  cout << "the value of a: " << p << endl;
13  cout << "address of a: " << &a << endl;
14  cout << "the value of p: " << p << endl;
15  cout << "the value pointed to by pointer p: " << *p << endl;
16  
17  return 0;
18}

指针类型所占的内存空间

  • 编译器编译的程序是32位时(X86)，指针类型占4个字节的内存空间；
  • 编译器编译的程序是64位时(X64, ARM64)，指针类型占8个字节的内存空间；

空指针和野指针

  • 空指针：指针变量指向内存中编号为0的空间；

    用途：初始化指针变量；

    注意：空指针指向的内存是不可以访问的；

  • 野指针：指针变量指向非法的内存空间；

  • 空指针和野指针都不是自己申请的空间，因此不能访问；

 1int main()
 2{
 3  //声明了一个空指针p1
 4  int * p1 = NULL;
 5  *p1 = 10;
 6  cout << "the value pointed to by pointer p1: " << *p1 << endl;
 7
 8  //给指针类型变量p2赋了一个地址值(int *)0x1100，
 9  //但该地址值不是本程序申请的，因此无法对该地址指向的内存空间进行读写
10  int * p2 = (int *)0x1100;
11  *p2 = 10;
12  cout << "the value pointed to by pointer p2: " << *p2 << endl;
13  
14  return 0;
15}

const修饰指针

const修饰指针有三种情况：

  1. const修饰指针 —常量指针（常量的指针，该指针指向的值是一个常量）；
  2. const修饰常量 —指针常量（该指针类型变量是一个常量）；
  3. const既修饰指针，又修饰常量；

常量指针 const int * p = &a;

  • 指针的指向可以修改；
  • 指针指向的值不可以修改；
 1int main()
 2{
 3  int a = 10;
 4  int b = 100;
 5  
 6  const int * p = &a;//声明了一个常量指针p
 7
 8  *p = 20;//对指针p指向的值进行修改，会报错
 9  p = &b;//改变指针p的指向，不会报错
10
11  return 0;
12}

指针常量 int * const p = &a;

  • 指针的指向不可以修改；
  • 指针指向的值可以修改；
 1int main()
 2{
 3  int a = 10;
 4  int b = 100;
 5  
 6  int * const p = &a;//声明了一个指针常量
 7
 8  *p = 20;//该指针常量指向的值可以修改
 9  p = &b;//该指针类型p是一个常量，因此p的值不能修改
10
11  return 0;
12}

const既修饰指针，又修饰常量

  • 指针的指向和指针指向的值都不可以修改；
 1int main()
 2{
 3  int a = 10;
 4  int b = 100;
 5  const int * const p = &a;//既修饰指针，又修饰常量
 6
 7  *p = 20;//指针p指向的值是一个常量，因此不可以修改
 8  p = &b;//指针p是一个常量，因此指针p的值即指针p的指向不可以修改
 9
10  return 0;
11}
